Everything is real

Every feature listed here actually ships.

No coming-soon banners, no mocked-up screenshots stitched together from a marketing brief. If it's described below, you can do it in the app you download from this page.

Schema browser

Tables, views, materialized views, functions, sequences across every schema on the connection.

  • Star tables — pinned Favorites and Recents sections
  • Per-kind toggles (tables / views / matviews / funcs / seqs)
  • Right-click for VACUUM ANALYZE, REINDEX, REFRESH MATVIEW, rename, drop, add column, add index
  • Copy schema-qualified name or generated CREATE script

SQL editor (Monaco)

Full Monaco editor with the keyboard shortcuts you already know — running locally, no CDN.

  • Run · run-selection · EXPLAIN ANALYZE (⌘E) · cancel running query
  • Auto-format (sql-formatter) on demand or on run
  • Per-tab draft autosaves across restarts
  • Query history, snippets, notices/warnings panel below results

Inline edit with diff review

Click to edit, multi-row select, clone, delete. Nothing hits the DB until you press Save.

  • Type-aware editors — bool dropdown, JSON textarea, plain input
  • NULL button (or ⌘⌫) inside the cell editor
  • EMPTY string renders distinct from NULL
  • Undo last delete before commit

Cell inspector

A right-side panel for the selected cell — type, current value, and the distinct values across that column.

  • Search-as-you-type across up to 200 distinct values
  • Click any value to filter the grid by it
  • Edit the value inline; the panel honors column type

Filters & sort that compose

Per-column filter row with the dozen ops you actually use, plus a raw WHERE field that ANDs with the rest.

  • = ≠ < ≤ > ≥ LIKE ILIKE IN NOT IN IS NULL IS NOT NULL
  • Active filters surface as dismissible chips above the grid
  • Multi-column sort (shift-click), with priority badges
  • ⌘F filters the rendered page client-side without re-querying

Foreign-key click-through

Single-column FKs get a jump button. Click it to open the referenced row in a new tab, prefiltered by PK.

  • ↗ badge on FK column headers
  • One-click jump opens the target row filtered by its key
  • Works across schemas

Copy & export — your way

Five formats from one menu, applied either to the selected rows or every visible row.

  • TSV, CSV, JSON, INSERT statements, Markdown
  • Save-as dialog for the full result
  • ⌘C copies selected rows as TSV by default

Real pagination

Page-size dropdown, jump-to-page input, first / prev / next / last. Auto-refresh on a 5/15/30/60-second interval.

  • Sizes 25 / 50 / 100 / 200 / 500 / 1000
  • Always shows total count alongside current window
  • "Show SQL" surfaces the exact SELECT driving the grid

Multi-connection workflow

Each connection opens in its own window with a color bar, breadcrumb, and live latency chip.

  • Paste a postgres:// URL — parsed into form fields
  • Copy current values back as a URL
  • SSL: disable / require / verify-full
  • Per-connection accent color so you never confuse prod and staging
